# 田地面积测量工具 - 规格文档

## 1. 项目概述
- **项目名称**: 田地面积测量工具
- **项目类型**: 移动端网页应用 (HTML/CSS/JS)
- **核心功能**: 通过GPS轨迹测量不规则地块面积
- **目标用户**: 农户、农业从业者

## 2. UI/UX 规格

### 布局结构
- **顶部状态栏**: 显示GPS状态、精度、点数
- **主显示区域**: 轨迹画布/结果显示
- **底部操作区**: 大尺寸操作按钮

### 响应式设计
- 适配移动端 viewport
- 按钮高度: 56px (便于触摸)
- 字体大小: 16px 最小值

### 视觉设计
- **主色调**: #2E7D32 (农业绿)
- **辅助色**: #4CAF50 (亮绿)
- **警告色**: #FF9800 (橙)
- **错误色**: #F44336 (红)
- **背景色**: #F5F5F5 (浅灰)
- **卡片背景**: #FFFFFF
- **字体**: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto

### 组件
1. **状态指示器**: GPS图标 + 精度显示 + 点数统计
2. **轨迹画布**: Canvas绘制轨迹多边形
3. **结果卡片**: 显示面积、周长、点数
4. **操作按钮**: 开始/结束/重置 (带图标)

## 3. 功能规格

### GPS轨迹采集
- 使用 Geolocation API, enableHighAccuracy: true
- 最小移动距离阈值: 3米
- 精度过滤: 只接受 accuracy < 10米 的点
- 实时显示: 当前坐标、点数量、GPS精度

### 面积计算
- 算法: 鞋带公式 (Shoelace formula)
- 单位转换: 1亩 = 666.6667平方米
- 输出: 平方米、亩 (保留2位小数)
- 额外输出: 周长、点数

### 交互流程
1. 页面加载 → 检测GPS可用性
2. 点击"开始测量" → 开始记录轨迹
3. 实时显示轨迹点和精度
4. 点击"结束测量" → 计算并显示结果
5. 可选择"重新测量"清除数据

### 异常处理
- GPS不可用: 提示打开权限/移动到开阔地带
- 点数<3: 提示无法构成多边形
- 精度过低: 显示警告提示

### 附加功能
- 模拟演示模式 (测试用)
- 轨迹可视化 (Canvas绘制)
